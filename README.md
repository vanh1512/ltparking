## Run project

```sh
cd reactjs
npm install 
```

And then start

```sh
npm start
```

## Install package if missing
```
leaflet: 
```
```sh
npm i leaflet
```
react-virtualized for typescript version 3.7 or lower: 

```sh
npm i @types/react-virtualized@9.21.15
```

## Deployment

```sh
npm build
```
